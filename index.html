3. BIM como Gestão

O processo BIM bem sucedido começa muito antes de qualquer modelagem, todas as informações necessárias para projetar devem ser concentradas em um BEP.

BEP DASHBOARD DE ARQUIVOS
DESCRIÇÃO GANHOS NECESSIDADES PRIORIDADE
BEP mínimo • Maior organização dos nossos arquivos, definição de nome e local de arquivo dês do início dos projetos

• Maior definição do nosso escopo, já que definir quais modelos serão feitos é definir escopo de projeto

• Possibilidade de acompanhar a cumprimento do documento		 • Inclusão da equipe BIM nas primeiras etapas de projeto para elaboração do documento

• Treinamento da equipe técnica para cumprimento do documento
BEP completo • Centralização de todas as informações necessárias para elaboração do projeto em um único documento • Desenvolvimento do BEP em um sistema interno que concentre todas as informações necessária para a elaboração dos nossos modelos.

• Colaboração dos setores responsáveis pelo projeto, dês da captação, documentos como ata de captação ou requerimento devem ser "absorvidos" pelo BEP, caso a informação deles sejam relevantes para elaboração dos modelos

• Necessários maiores estudos dos processos da empresa

4. Orçamento 100% Baseado em Modelo

Um excelente "termômetro" do nível de qualidade de um modelo BIM é o quão facilmente ele pode ser usando para a orçamentação do projeto.
Observação: Idealmente a orçamentação deve ser feita somente após a compatibilização de projeto que está no item 4.
DESCRIÇÃO GANHOS NECESSIDADES PRIORIDADE
Controle de qualidade dos modelos • Maior qualidade dos modelos

• Maior agilidade na resolução dos problemas apontados pelo setor de BIM		 • Maior capacitação da equipe de liderança para que possam verificar não apenas o projeto mas também a organização mínima do modelo e o atendimento a verificação BIM

 
Todos os itens de projeto definidos em modelo • Todos os itens anteriores

• Modelo federado 100% completo

• Maior precisão na extração de quantitativos

• Modelo fidedigno com a realidade, incluindo todos os elementos que precisam ser previstos para ser utilizados para orçamentação, acompanhamento de obras, etc		 • Inclusão de todos os elementos a serem quantificados em modelo

• Contratação de modelador BIM para o setor de estruturas que faça a modelagem de todos os itens que atualmente são feitos em CAD (como são projetos que já estão prontos em CAD pode aumentar o tempo de elaboração de elementos como escadas rampas, etc)

• Contratação de Especialista BIM com experiencia em orçamento  dentro dessa metodologia (sugerido atuar diretamente no setor de orçamento antes de trabalhar no BIM). Reestruturação do processo de orçamento
Orçamentação feita com o modelo • Todos os itens anteriores

• Maior agilidade na elaboração dos orçamentos		 • Contratação de Especialista BIM com experiencia em orçamento dentro dessa metodologia (sugerido atuar diretamente no setor de orçamento antes de trabalhar diretamente no BIM). Reestruturação do processo de orçamento

• Reestruturar o processo de orçamento utilizando o modelo BIM (necessário maiores estudos)

• Compra de uma ferramenta de orçamento voltada para modelos (OcaBIM, Visus Cost Management, PriMus, Volare, etc). Pode ser um desenvolvimento interno

DESCRIÇÃO GANHOS NECESSIDADES PRIORIDADE
Automações diversas dos nossos processos • Maior agilidade e qualidade na modelagem

• Criação de um sub-setor de plug-ins dentro do setor de BIM

• Rastreabilidade total do histórico de intervenções nos equipamentos

• Preenchimento rigoroso de parâmetros não-geométricos (fabricante, vida útil, data de instalação etc)
